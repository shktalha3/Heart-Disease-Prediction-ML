# Heart-Disease-Prediction-ML
Heart Disease Prediction System using Machine Learning and Streamlit. The Project Predicts the Likelihood of heart disease based on medical attributes and includes model training, evaluation, and a simple web interface
# ❤️ Heart Disease Prediction System

## 📌 Overview
This project is a Machine Learning-based system that predicts the likelihood of heart disease using patient medical data. It also includes a Streamlit web interface for interactive predictions.

---

## 🚀 Features
- Data preprocessing and cleaning
- Machine learning model training
- Model evaluation with accuracy (~85.25%)
- Streamlit web application for prediction
- Classification report analysis

---

## 🧠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

## 📂 Files Included
- app2.py → Streamlit web app
- heart_disease_prediction.py → ML model training
- heart(main).csv → Dataset
- requirements.txt → Dependencies
- Report PDF → Full project documentation

---

## 📊 Results
- Model Accuracy: **85.25%**
